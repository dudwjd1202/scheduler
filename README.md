# 🎙️ Lecture Calendar (과제 자동 등록 비서)

> **"다음 주 화요일까지 인공지능 레포트 제출하세요."**
> 교수님의 말씀을 듣고, 자동으로 과제 내용과 마감일을 캘린더에 등록해주는 안드로이드 앱입니다.

---

## 📅 프로젝트 소개 (Introduction)

강의 중 교수님이 구두로 공지하는 과제 마감 기한을 놓치거나, 캘린더에 일일이 입력하기 귀찮았던 경험에서 시작했습니다.
이 앱은 음성 인식(STT) 기술을 활용하여 강의 내용을 텍스트로 변환하고, **자체 알고리즘(NLP)** 을 통해 **'과제명'**과 **'마감 날짜'**를 자동으로 추출하여 스마트폰 캘린더에 등록합니다.

## 💡 핵심 기능 (Key Features)

1.  **음성 인식 (STT)**
    * `SpeechRecognizer`를 사용하여 실시간으로 한국어 음성을 텍스트로 변환합니다.
2.  **지능형 텍스트 분석 (Rule-based NLP)**
    * 복잡한 문장 속에서 **"~까지", "~제출"** 등의 키워드를 분석해 과제 제목을 추출합니다.
    * **"오늘", "내일", "다음 주 화요일"** 같은 상대적 날짜 표현을 **"2025-11-25"** 같은 실제 날짜 데이터로 자동 변환합니다.
3.  **캘린더 자동 연동**
    * 분석된 데이터를 바탕으로 스마트폰 기본 캘린더 앱을 호출(`Intent`)하여, 터치 한 번으로 일정을 저장합니다.
